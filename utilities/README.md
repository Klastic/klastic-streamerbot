# Utilities

Reusable helper scripts that can be called from other actions or used as standalone sub-actions.

| Utility | File | Description |
|---|---|---|
| Per-User Cooldown | [`user-cooldown/`](user-cooldown/) | Enforces per-user command cooldowns via user variables |
| Cross-Platform Follow Store | [`follow-store/`](follow-store/) | Read/write/query the JSON follow data file used by `!followage` for YouTube and Kick |
